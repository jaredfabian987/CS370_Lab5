Lab 5: Card Shuffler

This time we will learn displaying images using javax.swing.ImageIcon. You need
to display a lot of images on the GUI. First you need to manually download all
poker card images from some sources. You should display all the poker card
images on the main GUI, and a button at the bottom. Every time the button is
clicked, the poker cards should be shuffled and redisplayed. Use your creativity to
implement your GUI.
